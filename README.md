# SHOA_Titan_Memory
**Little Titan - experimental AI agent implementing SHOA principles**
## Features
- `запомни <name>: <code>` - learn a new ability
- `/run <name>` - execute a learned ability
- Web UI on port 5000
## Quick start
1. Install dependencies: `pip install flask requests psutil python-docx`
2. Run `python web_ui.py`
3. Open `http://127.0.0.1:5000`
